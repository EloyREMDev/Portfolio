# Portfolio Project

#"Este repositorio contiene mi portafolio personal, desarrollado completamente con HTML y CSS. Es un sitio web responsivo que presenta mis proyectos, habilidades y experiencia. El diseño se centra en una navegación intuitiva y un estilo moderno, utilizando buenas prácticas de desarrollo web. Ideal para mostrar mi trabajo y crecer profesionalmente como desarrollador web.

## 🌟 Descripción del Proyecto
 ### El proyecto incluye:

- **Sección de Introducción: Presentación personal con un diseño atractivo y responsivo.**

- **Sección de Habilidades: Visualización de habilidades técnicas usando imágenes y textos estilizados.**

- **Sección de Proyectos Recientes: Enlaces a repositorios y demostraciones en vivo de otros proyectos destacados.**

- **Pie de Página: Créditos al autor y recursos utilizados.**

## Estructura de Archivos
- **index.html: Archivo principal que organiza el contenido del sitio.**

- **index.css: Archivo de estilos que implementa diseño responsivo, colores personalizados y efectos visuales dinámicos.**

## 💡 Tecnologías Utilizadas
- **HTML5: Estructuración semántica del contenido.**

- **CSS3: Personalización avanzada de estilos, incluyendo:**

- **Uso de :root para definir variables temáticas.**

- **Diseños con flexbox para garantizar la alineación.**

- **Scrollbar personalizado con transparencia y estilos responsivos.**

- **Sombras y transiciones para dar un toque profesional.**
